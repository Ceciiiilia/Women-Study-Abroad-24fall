### 使用指南


| 板块 | 功能 | 特点 |
| --- | --- | --- |
| Issues | 【闲聊解压】【资源链接】【技巧工具】| 搬运资源 / 夸夸楼吐吐槽 |
| Discussions | 【每日打卡】【学习笔记】【交流讨论】 | 自己写的 / 经验交流 / 习惯养成 |


> 任何有用的网站、经验贴、工具等资源链接都可以po到issues对应楼里，也可以建新issue

> Discussions不做过多限制，发挥媎妹们的想象力与创造力！欢迎新建帖子分享与讨论，有用的干货会链接到本页！
